
	root 安装 root 运行 (sh ./install 默认完成后自动运行 daemon)
	
	log 可查看运行实时信息 tailf /usr/local/vc_linux_client/log/linux_client_log

	配置文件:/usr/local/vc_linux_client/etc/linux_client.xml	
	
	./kill_vc_client.sh 中止运行 daemon

	./run_vc_client.sh 启动服务

    ./uninstall.sh 终止并卸载程序
